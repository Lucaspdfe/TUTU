# Debugging

## How to install the debugger

The debugger (bochs) can be installed on your package manager of choice:

### For Ubuntu/Debian Systems:

    sudo apt install bochs vgabios bochs-sdl bochs-doc bochsbios

### For ArchLinux and derivatives:

    sudo pacman -S bochs bochs-sdl vgabios

## Debbuging:

To debug, you need to run:
    bochs -f bochs_config
